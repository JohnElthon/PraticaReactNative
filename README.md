# PraticaReactNative

Repositório de prática com exercícios e testes em React Native.

## Sobre o projeto

O **PraticaReactNative** foi desenvolvido durante meus estudos em React Native, com o objetivo de praticar conceitos vistos em aula e melhorar minha lógica no desenvolvimento de aplicativos mobile.

Este projeto faz parte da minha evolução como estudante de Desenvolvimento de Sistemas.

## Funcionalidades

- Criação de componentes
- Uso de estilos
- Prática de estrutura de projeto
- Testes de conceitos aprendidos em aula

## Tecnologias utilizadas

- React Native
- JavaScript
- Expo
- Node.js / npm

## Como executar o projeto

1. Clone este repositório:

```bash
git clone https://github.com/JohnElthon/PraticaReactNative.git
```

2. Acesse a pasta do projeto:

```bash
cd PraticaReactNative
```

3. Instale as dependências:

```bash
npm install
```

4. Execute o projeto:

```bash
npm start
```

5. Abra no emulador Android/iOS ou no aplicativo Expo Go.

## Observação

Projeto desenvolvido com fins acadêmicos e de aprendizado. Algumas partes podem representar práticas simples de aula, com foco em entender conceitos específicos do React Native.
